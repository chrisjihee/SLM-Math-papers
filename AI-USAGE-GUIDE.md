# KSC 논문 AI 도구 활용 가이드

## 📖 개요

KSC 2023-2024 우수논문 10편을 AI 도구(ChatGPT, Claude, Gemini, Cursor 등)에서 최적으로 활용하기 위한 가이드입니다.

## 📂 파일 구조

```
LM-based-KG-papers/
├── pdf-KSC/                    # 원본 PDF 및 DOCX 파일
├── md-KSC/                     # Markdown 변환 파일 ⭐
│   ├── README.md               # 논문 목록 및 구조 안내
│   └── [10개 논문].md          # 각 논문의 Markdown 버전
├── papars-KSC.yaml             # 논문 메타데이터 (초록, 키워드, 팁) ⭐
├── AI-USAGE-GUIDE.md           # 본 가이드 ⭐
└── convert_docx_to_md.py       # 변환 스크립트
```

## 🎯 AI 도구별 활용 방법

### 1. ChatGPT / Claude 활용

#### 방법 A: 개별 논문 업로드
```
1. md-KSC 폴더에서 관심 논문의 .md 파일 선택
2. ChatGPT/Claude에 파일 업로드
3. 다음과 같은 프롬프트 사용:

"위 KSC 논문을 분석하여 다음을 요약해주세요:
1. 핵심 연구 방법론
2. 주요 실험 결과
3. 내 연구 [주제명]에 적용 가능한 아이디어"
```

#### 방법 B: 여러 논문 동시 참조
```
papars-KSC.yaml 파일을 업로드하고:

"이 YAML 파일에는 KSC 우수논문 10편의 정보가 있습니다.
내 연구 주제는 [주제]인데, 가장 관련성 높은 논문 3편을 추천하고
각각에서 참고할 만한 방법론을 정리해주세요."
```

#### 방법 C: 논문 작성 지원
```
md-KSC 폴더의 여러 논문을 업로드하고:

"위 논문들의 서론 작성 스타일을 분석하여,
내 논문 제목 '[제목]'에 맞는 서론 초안을 작성해주세요.
특히 다음을 포함해주세요:
- 연구 배경 및 동기
- 기존 연구의 한계
- 본 연구의 기여점"
```

### 2. Cursor AI 활용

Cursor에서는 `@` 태그를 사용하여 파일이나 폴더를 참조할 수 있습니다.

#### 전체 논문 참조
```
@md-KSC 폴더의 논문들을 분석하여 
"데이터 증강" 관련 방법론을 정리해주세요
```

#### 특정 논문 참조
```
@md-KSC/고품질 인스트럭션 데이터 구축을 위한 피드백 기반 합성데이터 생성 방법.md
이 논문의 프레임워크를 참고하여 내 데이터 파이프라인 코드를 작성해주세요
```

#### YAML 메타데이터 활용
```
@papars-KSC.yaml
내 연구 키워드: ["LLM", "데이터 구축", "편향"]
관련 논문들의 핵심 팁(tips)을 정리해주세요
```

### 3. Gemini 활용

Gemini는 긴 문맥을 잘 처리하므로, 여러 논문을 한 번에 업로드할 수 있습니다.

```
[md-KSC 폴더의 10개 파일 모두 업로드]

"위 10편의 KSC 우수논문을 분석하여:
1. 공통적으로 사용된 연구 방법론 패턴
2. 각 논문의 독특한 기여점
3. 2025년 KSC 논문 작성 시 고려할 트렌드

를 정리해주세요."
```

## 💡 활용 시나리오별 가이드

```

프롬프트:

```

### 시나리오 2: 관련 연구 섹션 작성

```
프롬프트:
"위 논문들을 참고하여 내 논문의 '관련 연구' 섹션을 작성해주세요.

각 논문의 기여점과 한계를 언급하고, 
내 연구가 어떻게 차별화되는지 설명해주세요."
```

### 시나리오 3: 실험 설계 참고

```
[실험 방법론이 유사한 논문 2-3개 업로드]

프롬프트:
"위 논문들의 실험 설계를 분석하여:
1. 사용된 데이터셋과 전처리 방법
2. 평가 지표 및 baseline 설정
3. 실험 결과 제시 방식

을 정리하고, 내 연구 [주제]에 적용 가능한 실험 설계안을 제안해주세요."
```

### 시나리오 4: 논문 구조 설계


### 시나리오 5: 방법론 비교 분석

```
프롬프트:
"위 논문들에서 사용된 LLM 활용 방법론을 비교 분석하여 표로 정리해주세요.
컬럼: 논문 제목, 사용된 LLM, 활용 방식, 주요 성과"
```

## 📊 메타데이터 활용 (papars-KSC.yaml)

YAML 파일에는 각 논문의 다음 정보가 포함되어 있습니다:

- `abstract`: 논문 초록
- `keywords`: 핵심 키워드
- `contents`: 주요 내용 요약
- `relevance`: 연구 관련성
- `tips`: 실용적인 팁
- `md_file`: Markdown 파일 경로

### YAML 기반 검색
```
"papars-KSC.yaml에서 
keywords에 '데이터 증강'이 포함된 논문들의 
tips를 모두 정리해주세요"
```

## 🎓 주의사항

1. **인용 표기**: AI가 생성한 내용이라도 원 논문의 아이디어를 사용할 경우 반드시 인용 표기
2. **사실 확인**: AI 요약의 정확성을 원본 논문(PDF)로 검증
3. **창의성 유지**: AI는 참고용이며, 본인의 독창적인 아이디어가 핵심
4. **윤리 준수**: 표절 금지, 연구 윤리 규정 준수
