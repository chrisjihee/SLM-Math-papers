# KSC 우수논문 Markdown 모음

## 📋 개요

이 디렉토리는 KSC(한국소프트웨어종합학술대회) 2023-2024 우수논문 10편을 AI 도구가 쉽게 읽고 분석할 수 있는 Markdown 형식으로 변환한 것입니다.

## 🎯 목적

- **KSC 2025 논문 작성 참고**: 우수논문들의 내용과 형식을 분석하여 새로운 논문 작성에 활용
- **AI 도구 활용**: ChatGPT, Claude, Gemini 등 다양한 AI 모델이 논문 내용을 이해하고 참고할 수 있도록 최적화
- **구조화된 형식**: Markdown 형식으로 제목, 섹션, 표, 그림 등을 명확하게 구조화

## 📚 수록 논문 목록

### KSC 2024 (5편)

1. **한국어 수학 튜터링 대화 데이터셋**
   - 키워드: 대화 데이터셋, 수학 추론, 튜터링
   - 주요 내용: 수학 추론 능력 기반 튜터링 대화 데이터셋 구축

2. **한국어 생성문서의 사실성 평가에 대한 연구**
   - 키워드: 사실성 평가, 생성 문서, LLM
   - 주요 내용: 생성형 AI의 출력물에 대한 사실성 평가 방법론

3. **세분화된 LLM 피드백을 활용한 요약문 평가 모델 학습**
   - 키워드: LLM 피드백, 요약 평가, 모델 학습
   - 주요 내용: LLM의 세밀한 피드백을 활용한 요약문 품질 평가 모델

4. **생성형 AI의 암묵적 편향 완화를 위한 LLM 기반 학습데이터 자동 구축 방안**
   - 키워드: 편향 완화, 학습데이터, 윤리적 AI
   - 주요 내용: 생성형 AI의 암묵적 편향 탐지 및 완화를 위한 데이터 구축 방법론

5. **고품질 인스트럭션 데이터 구축을 위한 피드백 기반 합성데이터 생성 방법**
   - 키워드: 인스트럭션 튜닝, 합성 데이터, 피드백
   - 주요 내용: 피드백 루프를 활용한 고품질 인스트럭션 데이터 자동 생성

### KSC 2023 (5편)

6. **ChatGPT 기반 독해 튜터링 대화 데이터의 구축 타당성 조사**
   - 키워드: ChatGPT, 독해 튜터링, 데이터 구축
   - 주요 내용: LLM을 활용한 교육용 대화 데이터 구축의 타당성 검증

7. **대형 언어 모델을 활용한 퓨샷 추론 문제의 데이터 증강**
   - 키워드: 퓨샷 학습, 데이터 증강, ARC
   - 주요 내용: LLM을 활용한 추론 문제의 예제 데이터 자동 생성

8. **비한글 발음열 변환을 위한 통합 프레임워크: 숫자 및 영어단어 처리의 최적화**
   - 키워드: 발음 변환, TTS, 전처리
   - 주요 내용: 한국어 TTS를 위한 비한글 텍스트 발음 변환 최적화

9. **한국어 대규모언어모델을 활용한 MMR기반 자가생성 In-Context Learning**
   - 키워드: In-Context Learning, MMR, 자가생성
   - 주요 내용: 한국어 LLM의 In-Context Learning 성능 향상을 위한 예제 선택 방법

10. **ARCLE: 추상화 및 추론을 위한 강화학습 환경**
    - 키워드: 강화학습, ARC, 추상적 추론
    - 주요 내용: 추상적 추론 능력 평가를 위한 강화학습 환경 구축

## 🔍 논문 구조 분석

### 일반적인 KSC 논문 구조

```
# 제목
- 저자 정보
- 소속 기관

## 요약 (Abstract)
- 연구의 핵심 내용을 2-3 문단으로 요약

## 1. 서론 (Introduction)
- 연구 배경 및 동기
- 연구 목적 및 필요성

## 2. 관련 연구 (Related Work)
- 기존 연구 소개 및 비교
- 본 연구의 차별점

## 3. 제안 방법 (Proposed Method)
- 핵심 방법론 설명
- 시스템 아키텍처/프레임워크
- 알고리즘 상세

## 4. 실험 (Experiments)
- 실험 환경 및 데이터셋
- 실험 결과 및 분석
- 비교 실험

## 5. 결론 (Conclusion)
- 연구 결과 요약
- 한계점 및 향후 연구 방향

## 참고문헌 (References)
```

## 💡 AI 도구 활용 가이드

### ChatGPT/Claude에서 활용하기

```
이 논문들을 참고하여 다음과 같이 활용할 수 있습니다:

1. 논문 구조 참고
   "위 논문들의 구조를 분석하여 내 논문의 목차를 작성해주세요"

2. 연구 방법론 비교
   "위 논문들에서 사용된 LLM 활용 방법론을 비교 분석해주세요"

3. 관련 연구 정리
   "내 연구주제 [주제]와 관련된 논문들을 위 목록에서 찾아 요약해주세요"

4. 작성 스타일 학습
   "위 논문들의 서론 작성 스타일을 분석하고 내 논문의 서론 초안을 작성해주세요"
```

### 폴더 전체를 업로드하는 방법

1. ChatGPT/Claude: "md-KSC 폴더의 모든 파일을 읽고..." 로 시작하는 프롬프트 사용
2. Cursor: `@md-KSC` 태그를 사용하여 전체 폴더 참조

## 📝 주의사항

1. **그림 및 이미지**: 원본 docx 파일의 그림은 텍스트 설명으로만 포함되어 있습니다. 상세한 그림은 원본 PDF를 참고하세요.

2. **수식**: 일부 수식이 텍스트로 변환되어 정확하지 않을 수 있습니다. 수식이 중요한 경우 원본 PDF를 확인하세요.

3. **표**: 복잡한 표의 경우 Markdown 형식으로 변환되면서 구조가 단순화되었을 수 있습니다.

4. **인용**: 참고문헌 번호가 일부 누락되거나 형식이 변경되었을 수 있습니다.
