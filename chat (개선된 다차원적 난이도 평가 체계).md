## 🔄 **개선된 다차원적 난이도 평가 체계**

### 📊 **3가지 핵심 차원**

#### 1️⃣ **추론 깊이 (Reasoning Depth)**
- **Stage 1**: 1-step 직접 연산 (예: "사과 3개 + 바나나 2개 = ?")
- **Stage 2**: 2-3step 중간 추론 (예: "할인가격 계산 → 세금 적용 → 최종 금액")  
- **Stage 3**: 4+step 복합 추론 (예: "비율 분석 → 조건 분기 → 다단계 연산 → 검증")

#### 2️⃣ **개념 복잡성 (Conceptual Complexity)**
- **Stage 1**: 기본 사칙연산 (덧셈, 뺄셈, 곱셈, 나눗셈)
- **Stage 2**: 비율/분수/소수 조합, 단위 환산, 백분율
- **Stage 3**: 다변수 연립, 추상적 관계, 함수적 사고

#### 3️⃣ **혼동 요소 민감성 (Confounding Sensitivity)**
- **Stage 1**: 단순 정보 추출 (모든 정보가 해결에 필요)
- **Stage 2**: 부분적 misleading 정보 (일부 불필요한 정보 포함)
- **Stage 3**: 복잡한 distractor 패턴 (의도적 혼동 요소, 함정 정보)

### 🤖 **GPT-4 기반 자동 평가 시스템**

```prompt
이 수학 문제를 다음 3가지 차원에서 각각 1-5점으로 평가해주세요:

1. 추론 깊이: 이 문제를 풀기 위해 필요한 논리적 단계는 몇 개인가요?
2. 개념 복잡성: 관련된 수학적 개념들이 얼마나 추상적이고 복잡한가요?  
3. 혼동 가능성: 문제에 포함된 정보 중 혼동을 일으킬 수 있는 요소는 얼마나 많은가요?

[MWP 문제 텍스트]
```

**최종 난이도 = α×추론깊이 + β×개념복잡성 + γ×혼동민감성**
- 1.5/3.5 임계값으로 3단계 분할
- 각 stage 내 난이도 분산 최소화
- stage 간 명확한 구분성 확보
